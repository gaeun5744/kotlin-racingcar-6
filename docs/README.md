# 구현할 기능 목록

### 1. 자동차 이름 입력
- 입력 안내 메시지 출력
- 잘못 입력했을 경우 `IllegalArgumentException` 발생
    - string에 쉼표가 포함되어 있지 않은 경우
    - 각 이름이 5자 이하가 아닌 경우

### 2. 자동차 생성 
- 사용자 입력에 따라 이름이 부여된 자동차 객체 생성

### 3. 시도 횟수 입력
- 입력 안내 메시지 출력
- 숫자가 아닐 경우 `IllegalArgumentException` 발생

### 4. 전진 여부 결정
- 0과 9 사이의 무작위 값 생성
- 해당 무작위 값이 4 이상인지 확인
- 4 이상일 경우 해당 자동차에 - 추가

### 5. 진행 상황 및 결과 출력
- 시도 횟수 만큼 `4. 전진 여부 결정`을 반복하고 진행 상황 출력
- -가 가장 많은 우승자(들)을 마지막에 출력
